# Portafolio 

Bienvenido a mi portafolio personal.
Este sitio está desarrollado con Astro, diseñado con un estilo cyberpunk minimalista, y presenta mis proyectos, experiencia y habilidades como desarrollador Full Stack.

## Tecnologías principales

* Astro — Framework web ultrarrápido
* TailwindCSS — Estilos utilitarios
* TypeScript / JavaScript
* Componentes Astro + integración con íconos y animaciones


## Comandos
Comando	Acción

npm run dev	Arranca el servidor local (localhost:4321)
